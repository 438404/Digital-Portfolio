What Is Unit 13?

Unit 13 is website development, this being said creating a website to a given scenario. In the scenario we were given, we are working as a junior web developer creating a website for a phone company called 'ICUPhones'. With this task of creating a website, this means that we must take into consideration user requirements and website constrictions. For this we must create: A storyboard (outline the layout of the website), a site map (a structure of the directory) & a test plan to describe different tests we must conduct on the website to find its strengths and weaknesses and how we can improve the functionality of the website.
