Microsoft Specialist Skills

Microsoft Specialist Skills is a seperate subject we are engoing in College. For this, we use a program called 'GMetrix'. This is a program where we able to choose between 4 different Microsoft products: Word, Powerpoint, Excel & Outlook. When we choose the correct program, we are aligated to answer questions relating to different features of the program wether it may be: move, resize, delete or export. After we advance through each of the programs, we take a final exam & upon passing we are given a Specialist certificate signed by a Microsoft Employee themselves.
